# 표지 배경 생성 — 품질 검증 spike

이 서비스의 **단일 최대 리스크**는 코드가 아니라 이것 하나다:

> 조감도 1장을 넣었을 때, gpt-image-1이 **글자 없고 / 안전영역 확보된 / 톤·무드가 반영된** "PPT에 바로 붙일 수 있는" 표지 배경을 실제로 뽑아내는가?

이 spike는 본격 개발(인증·크레딧·결제) 전에 그 답을 하루 안에 눈으로 확인하기 위한 도구다.
통과하면 MVP로 진행, 실패하면 접근법(모델·프롬프트·후처리)을 먼저 고친다.

## 실행

```bash
cd cover-generator/spike
npm install
export OPENAI_API_KEY=sk-...          # 본인 키
node generate.mjs --input ./조감도.jpg # 본인 현장 조감도
```

- 결과 이미지는 `out/`에 저장되고, `out/index.html`에 비교 그리드가 만들어진다.
- API 호출 없이 프롬프트/비용만 보려면: `node generate.mjs --dry-run`

### 옵션

| 옵션 | 값 | 기본 |
|---|---|---|
| `--input` | 조감도 경로 (jpg/png/webp) | (필수) |
| `--quality` | `low` \| `medium` \| `high` | `medium` |
| `--ratio` | `a4-landscape` \| `16:9` \| `a4-portrait` | `a4-landscape` |
| `--out` | 출력 폴더 | `./out` |
| `--dry-run` | API 미호출, 프롬프트/비용만 | off |

기본 매트릭스는 6개 대표 조합(navy/gold/white/black/blue × 무드/밀도/안전영역)이다.
전 조합(630개)은 과하므로 대표만 본다. 조합은 `prompts.mjs`의 `DEFAULT_MATRIX`에서 조정.

## 결과에서 확인할 것 (go/no-go 기준)

1. **글자/로고 누출** — 가짜 한글·영어·워터마크·서명이 없는가? (가장 중요)
2. **안전영역** — 선택한 좌/중/우/하단이 실제로 차분하게 비워졌는가?
3. **톤·무드·밀도** — navy는 navy답게, minimal은 여백답게 나오는가?
4. **원본 유지** — 업로드한 조감도가 알아볼 수 있게 반영됐는가?

> 참고: 최종 서비스에서 안전영역은 프롬프트가 아니라 **`sharp` 후처리로 그라데이션을
> 결정론적으로 합성**할 계획이다. 이 spike는 "AI가 애초에 텍스트 없는 배경을 뽑는가"를
> 보는 게 1순위이고, 프롬프트만으로 안전영역이 어디까지 되는지는 참고용으로 함께 본다.

## 비용 주의

`medium` 기준 장당 약 $0.06, 6장이면 약 $0.4 (추정치). `high`는 장당 약 $0.25로 훨씬 비싸다.
이 단가가 곧 unit economics다 — "비회원 무료 생성"을 열어두면 그대로 비용으로 샌다.
